# TM Digital Space - Agency Website

A high-converting, modern digital agency website built with Next.js 15, TypeScript, Tailwind CSS 4, Framer Motion, and GSAP.

## 🚀 Features

### Core Features
- ✅ **Modern Tech Stack**: Next.js 15 with App Router, TypeScript, Tailwind CSS 4
- ✅ **Smooth Animations**: Framer Motion + GSAP for professional micro-interactions
- ✅ **Responsive Design**: Mobile-first approach, perfect on all devices
- ✅ **Performance Optimized**: Fast loading, optimized images, code splitting
- ✅ **SEO Ready**: Proper meta tags, semantic HTML, structured data

### Sections
1. **Hero Section** - Eye-catching hero with animated background, CTAs, and trust indicators
2. **Services Section** - Showcasing 4 main services with detailed features
3. **Portfolio Section** - Filterable portfolio with before/after metrics and testimonials
4. **Why Choose Section** - Animated counters and feature highlights
5. **Process Section** - 4-step process with beautiful visualizations
6. **Testimonials** - Real client testimonials with ratings and results
7. **Contact/CTA Section** - Comprehensive contact form with multiple contact methods

### Advanced Features
- 🎨 **Dark/Light Mode Toggle** - Smooth theme switching
- 📱 **WhatsApp Quick Connect** - Floating WhatsApp button for easy communication
- 📊 **Scroll Progress Indicator** - Visual feedback as users scroll
- ⬆️ **Scroll to Top Button** - Easy navigation back to top
- 🎭 **Loading Animation** - Professional page loader
- 🖱️ **Micro-interactions** - Hover effects, button animations, smooth transitions

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
